You are a senior Python engineer. Build a complete, production-quality MVP that extracts item names and prices from PDF files and outputs them into a structured Excel file.

## 🎯 Goal
Create a Python app that:
1. Accepts a PDF file as input
2. Extracts item descriptions and prices
3. Outputs a clean Excel (.xlsx) file with structured columns

---

## 🧱 Tech Stack
- Python 3.11+
- pdfplumber (primary PDF parsing)
- PyMuPDF (fitz) as fallback parser
- pandas (data handling)
- openpyxl (Excel output)
- regex (price extraction)
- optional: Streamlit for UI

## ⚙️ Functional Requirements

### 1. PDF Reading (pdf_reader.py)
- Use pdfplumber to extract:
  - Tables if present
  - Raw text if no tables found
- Fallback to PyMuPDF if pdfplumber fails
- Return structured raw data (list of rows or text lines)

---

### 2. Price Extraction (price_parser.py)
- Identify prices using regex patterns like:
  - R 123.45
  - 123.45
  - 1,234.56
- Extract:
  - item name (text before price)
  - price (numeric)
- Handle messy lines like:
  "Milk 2L R23.99"
  "Bread White 700g 18.50"

Return structured list:
[
  {"item": "Milk 2L", "price": 23.99},
  {"item": "Bread White 700g", "price": 18.50}
]

---

### 3. Data Cleaning (cleaner.py)
- Remove duplicates
- Strip whitespace
- Normalize price to float
- Remove invalid rows

---

### 4. Excel Output (writer.py)
- Use pandas + openpyxl
- Output file: output/result.xlsx
- Columns:
  - Item Name
  - Price
- Auto-adjust column widths

---

### 5. Main Script (main.py)
- Accept file path as input argument
- Run pipeline:
  PDF → Extract → Parse → Clean → Excel
- Print progress logs
